===========================================================

HostInfo Example.

Utility that uses standard TCP/IP gethostbyX() functions
to retrieve information for a given host.

Pass the server name on the command line. Information is
printed to stdout.

Example:

HostInfo www.idgbooks.com

===========================================================

HOSTINFO.C		Source code. 
			Compile and link with WSOCK32.LIB

HOSTINFO.EXE	Compiled program.

===========================================================
